## Prerequisites

- [Docker](https://www.docker.com/get-started) installed on your machine

## Features

- Microservices architecture
- Secure authentication with JWT (normal users and admin)
- Visa services API
- Admin and user-based access control
- Built with .NET, Ocelot, and PostgreSQL

## Installation

1. Clone this repository to your local machine:

   ```bash
   https://github.com/Dcoders-CeylonScape/CeylonScape-Backend.git
   ```

2. Navigate to the project directory:
   ```bash
   cd CeylonScape-Backend
   ```

3.run command
   ```bash
   docker compose up --build
   ```
## Usage

Once the services are up, the application will be running locally. You can access the APIs through the configured gateway.
